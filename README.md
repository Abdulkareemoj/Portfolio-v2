## Running Locally

```bash
git clone https://github.com/Abdulkareemoj/Portfolio-v2
cd 
yarn
```

Create a `.env` file similar to [`.env.example`](https://github.com/jahirfiquitiva/jahir.dev/blob/main/.env.example).

```bash
yarn dev
```
